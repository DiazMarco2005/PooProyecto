-------------- Proyecto Horas Beca-----------------



Este proyecto tiene como objetivo una app de dispositivo móvil en el cual se tomará en cuenta la cantidad de cada alumno se hará un request a la base de datos y se mostrará la cantidad de horas beca. Todo esto será por medio de una interfaz gráfica "React Native" Utilizando para el backend JAVA y aplicación de clases vistas en Programación Orientada a Objetos.
